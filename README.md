| Button | What It Does | Real-World Technique |
|--------|-------------|----------------------|

> рҹ§  **PE Stomping Insight:** Advanced malware (Cobalt Strike, Metasploit Reflective DLLs) wipes `MZ` from RAM *after* loading, because EDR scanners search for `MZ` during live memory scans. No `MZ` = invisible.
